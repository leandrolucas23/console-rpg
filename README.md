# Terminal Text RPG

A text-based RPG developed in Python that runs entirely in the terminal, using only Python's standard library.

---

## About

This project is a console RPG where the player progresses through battles, exploration, dialogues, and decisions using text commands.

The goal of this project is to practice object-oriented programming, game logic, and Python fundamentals without relying on external libraries.

---

### Implemented

- [x] Character creation
- [x] Turn-based combat
- [x] Enemy encounters
- [x] Skills and abilities

## Getting Started

### Requirements

- Python 3.10 or newer

### Clone the repository

```bash
git clone https://github.com/leandrolucas23/console-rpg.git
```

### Navigate to the project folder

```bash
cd console-rpg
```

### Run the game

```bash
python3 game/main.py
```
